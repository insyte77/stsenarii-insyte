# CLAUDE.md — Правила проекта «Сценарии INSYTE»

## Режим работы

**Полная автономия.** Claude работает самостоятельно без запросов подтверждения.

### Разрешено без вопросов:
- Открывать и выполнять команды в PowerShell и Bash
- Создавать, редактировать, удалять файлы проекта
- Искать информацию в интернете (WebSearch, WebFetch)
- Делать git commit, git push
- Создавать и обновлять репозитории GitHub
- Обновлять настройки VS Code (Project Manager, settings.json)
- Записывать в файлы памяти

### Запрещено:
- Удалять репозитории GitHub
- Удалять системные файлы Windows
- Запрашивать подтверждение у пользователя (если только это не удаление репозитория)

## О проекте

Разработка технических заданий и сценариев для ИИ-блока автоматизированного создания сценариев.

**Контекст:** Это проектная часть INSYTE — создание инструментария на основе ИИ.

## Структура файлов

- `scenarios/` — готовые сценарии
- `templates/` — шаблоны ТЗ
- `specs/` — технические задания
- `memory/` — файлы памяти проекта

## Журнал диалогов

Все ключевые договорённости, правила и контекст диалогов сохраняются в `memory/MEMORY.md` и отдельных файлах памяти.
