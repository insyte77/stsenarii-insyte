Проект «Сценарии INSYTE» — проектная часть разработки автоматизированного ИИ-блока для создания сценариев.

**Why:** Нужен инструментарий для систематизации и автоматизации создания сценариев с помощью ИИ. Это техническая и проектная документация.

**How to apply:** При работе с проектом фокусироваться на:
- Создании технических заданий (ТЗ) для ИИ-сценариев
- Шаблонах и паттернах сценариев
- Структурировании готовых сценариев
- Документации для автоматизации

**Структура:**
- `scenarios/` — готовые сценарии
- `templates/` — шаблоны ТЗ
- `specs/` — технические задания
- `memory/` — файлы памяти проекта
