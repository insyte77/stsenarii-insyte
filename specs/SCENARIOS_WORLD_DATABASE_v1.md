### 4.4 Интеграция с котлом (OpenTherm)

OpenTherm — цифровой протокол управления газовым котлом. Позволяет:
- Плавная регулировка мощности котла (не просто Вкл/Выкл)
- Экономия газа до 15-25% по сравнению с реле-управлением
- Чтение параметров котла: ошибки, давление, температура
- Погодозависимое управление — котёл адаптирует T теплоносителя к уличной T

## 10. КОМПЛЕКСНЫЕ (МАСТЕР-) СЦЕНАРИИ

Это самые мощные сценарии — они одновременно управляют несколькими системами.

### 10.1 «Доброе утро» (Good Morning)

**Триггер:** Будильник / расписание / кнопка  
**Последовательность:**
1. За 30 мин до подъёма — шторы медленно приоткрываются (10-20%)
2. За 20 мин — свет начинает плавно нарастать с 1800K → 3000K (имитация рассвета)
3. За 10 мин — температура в спальне поднимается с 17 до 21°C
4. В момент подъёма — свет 60-70%, 4000K
5. В кухне включается свет, кофемашина (если подключена)
6. Открываются шторы в гостиной
7. На TV/планшете включается новостной канал / прогноз погоды
8. Вентиляция переходит в дневной режим  

**Польза:** Пробуждение без стресса, тело само просыпается, всё готово

---

### 10.2 «Я ухожу» / «Покинуть дом» (Leave Home)

**Триггер:** Кнопка у двери / геофенсинг / голос  
**Последовательность:**
1. Выключается весь свет
2. TV, проектор, медиаплеер — выключить
3. Кондиционеры — выключить
4. Отопление → эконом-режим (15-16°C)
5. Все шторы → положение «нейтральное» или закрыть
6. Проверить окна (датчики) — уведомление если открыты
7. Вентиляция → минимальный режим
8. Охрана → режим «Ушёл»
9. Ворота/гараж → закрыть
10. WiFi → только необходимые устройства

**Не выключается:** Холодильник, антизамерзание, датчики безопасности, пожарная система, видеонаблюдение, охрана.  
**Польза:** Экономия 30-40%, безопасность, спокойствие — одна кнопка всё решает

---

### 10.3 «Добро пожаловать домой» (Welcome Home)

**Триггер:** Геофенсинг (дом в X км) / RFID / домофон  
**Последовательность:**
1. Ворота открыть
2. Освещение входной зоны и коридора включить
3. Охрана → режим «Дома»
4. Отопление → комфортный режим (заранее)
5. Вентиляция → дневной режим
6. Любимая музыка (опционально)
7. Рольставни → положение «дома»

**Польза:** Дома всё готово — тепло, открыто, освещено

---

### 10.4 «Спокойной ночи» (Good Night)

**Триггер:** Кнопка в спальне / расписание / голос  
**Последовательность:**
1. Весь свет в нежилых комнатах → OFF
2. В спальне → плавное угасание за 15-20 минут
3. Шторы в спальне → 100% блэкаут
4. Все AV-устройства → OFF
5. Охрана → режим «Ночь» (периметр)
6. Температура спальни → 17-19°C
7. Вентиляция → ночной бесшумный режим, CO₂ < 700 ppm
8. Газовый кран на кухне → закрыть (опционально)
9. WiFi → отключить (ARC: ЭМИ-защита)
10. Ночной маршрут → активировать (подсветка пути в туалет)

**Польза:** Идеальные условия для сна, одна кнопка заменяет обход всего дома

---

### 10.5 «Тревога» (Alarm)

**Триггер:** Датчик движения / датчик открытия / кнопка тревоги  
**Последовательность:**
1. Сирена → ON (внешняя и внутренняя)
2. Весь свет → 100% (все комнаты)
3. Видеозапись → начало с всех камер
4. Уведомление на телефон с видео
5. Уведомление в охранную компанию
6. Ворота → заблокировать
7. Видео с камер → отправить в облако

**Польза:** Максимальное отпугивание, полная документация

---

### 10.6 «Пожар» (Fire)

**Триггер:** Датчик дыма / датчик тепла / кнопка  
**Последовательность:**
1. Все двери → разблокировать (эвакуация)
2. Весь свет → 100%
3. Вентиляция → СТОП (не раздувать огонь)
4. Кондиционеры → СТОП
5. Газовый клапан → закрыть
6. Сирена + голосовое «Пожар, покиньте здание»
7. Вызов 112 (если интегрировано)
8. Уведомление всем жильцам и владельцу
9. Ворота → открыть (для пожарных)
10. Запись видео со всех камер

**Польза:** Спасает жизни, автоматически делает всё правильно

---

### 10.7 «Режим гостя» (Guest Mode)

**Триггер:** Активация через приложение  
**Последовательность:**
1. Гостевой код доступа → создать на X дней
2. Гостевая комната → нормальный режим
3. Личные зоны → ограничить доступ (хозяйская спальня, кабинет)
4. Управление: только свет, климат, ТВ в гостевой зоне
5. Журнал: фиксировать входы/выходы гостя

**Польза:** Приватность хозяина, гость управляет своей зоной

---

### 10.8 «Имитация присутствия» (Away/Vacation Mode)

**Триггер:** Активация перед отпуском  
**Действие:**
- Случайное включение/выключение света в разных комнатах в вечернее время
- Периодическое открытие/закрытие штор
- Включение TV случайно
- Орошение сада по расписанию
- Охрана → максимальный режим

**Польза:** Дом выглядит жилым, взломщики уходят

---

### 10.9 «Романтика» (Romance Mode)

**Действие:**
- Свет → очень тёплый 1800K, 15-20% яркости только декоративный
- Шторы → закрыть
- Электрокамин → включить (если есть)
- Музыка → выбранный плейлист, тихий
- TV → OFF

**Польза:** Атмосфера в одно касание

---

### 10.10 «Вечеринка» (Party Mode)

**Действие:**
- Свет → динамический RGB или фиксированный весёлый цвет
- Музыка → громко, все зоны
- Климат → усиленная вентиляция (много людей)
- Охрана → временно снять с периметра (гости ходят)
- Ворота → открыть по расписанию

---

### 10.11 «Дети дома одни» (Kids Home Alone)

**Действие:**
- Определённые зоны недоступны (хозяйская спальня, кабинет, бар)
- Ограничение времени TV (например, до 21:00)
- Свет в детской → автоматически в 21:30 к сну
- Уведомление родителям при открытии входной двери
- Запрет выезда ворот

**Польза:** Контроль без физического присутствия
