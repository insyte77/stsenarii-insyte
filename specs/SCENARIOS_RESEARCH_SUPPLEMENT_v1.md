## ДОПОЛНИТЕЛЬНЫЕ СЦЕНАРИИ ИЗ ИССЛЕДОВАНИЯ

### Ступени — каскадная подсветка
- Подсветка каждой ступени включается последовательно при движении
- Направление: снизу-вверх при подъёме, сверху-вниз при спуске
- Ночной режим: постоянная тихая подсветка 10-20%
- «Водопад» — цветовые волны для эстетики

### Жалюзи — дополнительные сценарии
- **Office anti-glare**: жалюзи позиционируются под углом 45° для минимизации блика на мониторе
- **Тепловая масса**: зимой открыты днём для пассивного солнечного обогрева
- **Peak rate avoidance**: летом закрыты в часы пика тарифов для снижения нагрузки кондиционирования

### Умный гараж — дополнительно
- Auto-close через 2 мин после выезда автомобиля (датчик)
- Night security close: автозакрытие в 22:00 если не закрыт
- Status alert: уведомление если открыт >10 минут

### Ирригация — дополнительно
- Smart irrigation saves 25-45% воды vs. фиксированного расписания
- Sensor: soil moisture drops below 30% → полив

### Горячая вода — экономия
- Преднагрев к расписанию (6:00 и 18:00)
- Отключение при отсутствии >6 часов
- Снижение до 45°C ночью (vs. 55°C днём) при сохранении безопасности
- Интеграция с солнечным коллектором

## ТЕХНОЛОГИИ ОБНАРУЖЕНИЯ ПРИСУТСТВИЯ

### mmWave / 24GHz радар
- Обнаруживает микродвижения (дыхание, сидение)
- Не нужно двигаться для подтверждения присутствия
- Предотвращает нежелательное выключение света при неподвижном человеке
- Определяет: человек стоит / сидит / лежит / спит

### PIR (инфракрасный) — ограничения
- Требует движения для срабатывания
- Не видит через стекло
- Чувствителен к температуре окружения

### Комбинация PIR + mmWave — оптимальное решение
- PIR: срабатывание при приходе
- mmWave: подтверждение присутствия пока человек неподвижен

## КВАРТИРА VS. ДОМ — СПЕЦИФИКА

### Квартира (ограничения арендатора / МКД)
- Нельзя менять термостат, двери, окна
- Предпочтение: временные WiFi-устройства без монтажа
- Меньше устройств → простота важна
- Нет зон полива, сауны, ворот

### Дом (полный потенциал)
- Проводная интеграция (KNX, INSYTE шина) — максимальная надёжность
- Зональное отопление — все типы
- Ирригация, сауна, бассейн, ворота
- Периметр, ландшафтное освещение
- Сложная автоматизация без ограничений
