# Сценарии INSYTE

Проект для разработки технических заданий и сценариев автоматизированного ИИ-блока создания сценариев.

## Структура проекта

```
Сценарии INSYTE/
├── scenarios/          # Готовые сценарии
├── templates/          # Шаблоны ТЗ
├── specs/              # Технические задания
├── memory/             # Память проекта (правила, контекст)
└── CLAUDE.md           # Правила работы Claude в этом проекте
```

## Назначение

Проектная часть разработки системы автоматизированного создания сценариев с помощью ИИ:
- Сценарии для автоматизации
- Технические задания для ИИ-блоков
- Шаблоны и паттерны сценариев
